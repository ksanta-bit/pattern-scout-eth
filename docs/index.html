① Crea il token su GitHub
Nella pagina: alla voce Expiration scegli "No expiration" (così vale per sempre), poi scorri in fondo, premi Generate token e copia il codice (inizia con ghp_).

Le scelte lanciano il bot con le tue impostazioni e restano attive anche per i giri automatici.
Senza token? Apri GitHub → Actions → Run workflow (sei già loggato sul telefono) e scegli lì le opzioni.
